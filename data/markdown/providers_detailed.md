## How to Use This Directory

This directory contains detailed information about all ToggleHealth providers organized by city and provider type.

**Key Information for Each Provider:**
- **Provider ID:** Unique identifier for the provider
- **Networks:** Which ToggleHealth networks include this provider
- **Accepted Plans:** Specific plans that can be used with this provider
- **Contact Information:** Address, phone, and fax
- **Credentials:** Education, specialties, and experience

**Network Codes:**
- `TH-HMO-PRIMARY` = Gold HMO Plan
- `TH-PPO-PREMIER` = Platinum PPO Plan (all providers accept this)
- `TH-EPO-SELECT` = Silver EPO Plan
- `TH-HDHP-CORE` = Bronze HDHP Plan
