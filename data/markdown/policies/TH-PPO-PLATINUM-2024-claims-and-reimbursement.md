## Claims and Reimbursement

### In-Network Claims

When you see in-network providers, they will file claims directly with ToggleHealth. You are responsible only for copays, coinsurance, and deductibles.

### Out-of-Network Claims

When you see out-of-network providers, you may need to file a claim for reimbursement:

**How to File a Claim:**
1. Pay the provider in full
2. Request an itemized bill
3. Complete a claim form (download at my.togglehealth.com)
4. Submit to: ToggleHealth Claims Department, PO Box 98234, Seattle, WA 98109

**Required Documentation:**
- Itemized bill or receipt
- Provider name and tax ID
- Date of service
- Diagnosis code (ICD-10)
- Procedure code (CPT)
- Member ID number

**Claim Filing Deadline:** 365 days from date of service

**Processing Time:**
- Electronic claims: 14-21 days
- Paper claims: 30-45 days

### Appeals Process

If your claim is denied, you have the right to appeal:

**First Level Appeal:**
- Timeframe: Must file within 180 days of denial
- Address: ToggleHealth Appeals, PO Box 98235, Seattle, WA 98109
- Response Time: 30 days

**Second Level Appeal:**
- Timeframe: Must file within 60 days of first level denial
- Review Type: Independent external review
- Response Time: 45 days
