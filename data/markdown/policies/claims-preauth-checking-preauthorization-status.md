## Checking Preauthorization Status

### Online Status Check

**Website:** my.togglehealth.com/preauth-status

**Information Needed:**
- Member ID
- Reference number (provided when request submitted)

**Status Options:**
- Pending - Under review
- Approved - Approved with authorization number
- Denied - Not approved
- More Information Needed - Additional documentation required

### Phone Status Check

**Call:** 1-800-TOGGLE-2

**Available:** Monday-Friday, 8 AM - 6 PM Pacific

**Have Ready:**
- Member ID number
- Reference number or date of request
